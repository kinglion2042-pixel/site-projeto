Carro Arduino Desviador de Obstáculos

Um robô inteligente que detecta obstáculos, recua automaticamente, muda de direção e continua seu caminho. Projeto ideal para aprender robótica, programação e eletrônica com Arduino.

Como esse Projeto Funciona?

Este carro robótico utiliza um sensor ultrassônico para medir a distância de objetos à sua frente. Quando detecta um obstáculo muito próximo, ele automaticamente:

1. Para os motores e dá ré por um curto período

2. Usa um servo motor para girar o sensor e verificar a melhor direção

3. Vira para uma nova direção livre de obstáculos

4. Continua seguindo em frente até encontrar outro obstáculo

O controle principal é feito por um Arduino UNO, que processa os dados do sensor e comanda os motores através de um driver L298N.

Montagem Passo a Passo

1

Passo 1: Montagem da Base

Comece montando a estrutura básica do carro robô. Fixe os motores DC nas laterais da base utilizando os parafusos fornecidos. Certifique-se de que os motores estão firmemente presos e alinhados. Em seguida, instale as rodas nos eixos dos motores. Adicione a roda livre na parte dianteira ou traseira para garantir estabilidade.

2

Passo 2: Instalação do Arduino e Shield

Conecte o Arduino UNO ao Adafruit Motor Shield. Fixe essa combinação na parte superior da base do carro. O Motor Shield será responsável por controlar a velocidade e direção dos 2 motores DC. Conecte a fonte de alimentação (bateria Li-Ion) ao shield, tomando cuidado com a polaridade.

3

Passo 3: Conexão dos Motores e Sensor

Conecte os fios dos 2 motores DC às saídas do Motor Shield (M1 e M2). Instale o sensor ultrassônico HC-SR04 na parte frontal do carro, montado no servo motor para permitir varredura. Conecte o sensor ao Arduino: VCC no 5V, GND no GND, Trig no pino A0 e Echo no pino A1.

4

Passo 4: Instalação do Servo Motor

Monte o servo motor SG90 na parte dianteira do carro, conectado ao suporte do sensor ultrassônico. O servo permitirá que o sensor gire para os lados para verificar a melhor direção. Conecte o servo ao Arduino: fio vermelho no 5V, marrom no GND e laranja no pino 10. Teste o movimento do servo com um código básico antes de prosseguir.

5

Passo 5: Programação do Arduino

Conecte o Arduino ao computador via cabo USB. Abra a IDE do Arduino e crie um novo sketch. Instale as bibliotecas necessárias: AFMotor, NewPing e Servo. Programe a lógica: leitura do sensor ultrassônico, controle dos 2 motores através do Motor Shield, e movimento do servo para verificar direções. Faça upload do código para o Arduino.

6

Passo 6: Testes e Ajustes Finais

Desconecte o Arduino do computador e alimente o sistema com a bateria. Teste o funcionamento do carro em uma área livre com obstáculos. Observe se ele detecta obstáculos corretamente, dá ré, verifica as direções e muda de direção. Ajuste parâmetros no código como distância de detecção, velocidade dos motores e ângulo de virada. Faça os ajustes necessários até o carro funcionar perfeitamente.

Componentes Utilizados

Arduino UNO

Microcontrolador principal que processa os dados do sensor e controla os motores.

Sensor Ultrassônico HC-SR04

Detecta obstáculos medindo a distância através de ondas sonoras.

Servo Motor SG90

Controla a direção do sensor ultrassônico para verificar obstáculos laterais.

Adafruit Motor Shield

Controla a velocidade e direção dos dois motores DC do carro robô.

Motores DC (x2)

Fornecem a tração para o carro se mover para frente, trás e fazer curvas.

Bateria Li-Ion 2200mAh 3,7V

Fonte de energia para todo o sistema (Arduino, motores e sensores).

Cabos Dupont Jumper

Conexões para ligar todos os componentes entre si e ao Arduino.

Código do Carro Robô

Código Arduino Completo

Código completo para controlar o carro robô desviador de obstáculos. Inclui leitura do sensor ultrassônico, controle dos 2 motores e servo motor. Antes de usar, instale as bibliotecas necessárias: AFMotor, NewPing e Servo.

// CARRO ARDUINO DESVIO DE OBSTÁCULOS
// Antes de enviar o código, você precisa instalar as bibliotecas necessárias //
// Biblioteca AFMotor https://learn.adafruit.com/adafruit-motor-shield/library-install //
// Biblioteca NewPing https://github.com/livetronic/Arduino-NewPing //
// Biblioteca Servo https://github.com/arduino-libraries/Servo.git //
// Para instalar as bibliotecas vá em: Sketch >> Incluir Biblioteca >> Adicionar arquivo .ZIP >> Selecione os arquivos ZIP baixados pelos links acima //

#include <AFMotor.h>  
#include <NewPing.h>
#include <Servo.h> 

#define TRIG_PIN A0 
#define ECHO_PIN A1 
#define MAX_DISTANCE 200 
#define MAX_SPEED 190 
#define MAX_SPEED_OFFSET 20

NewPing sonar(TRIG_PIN, ECHO_PIN, MAX_DISTANCE); 

AF_DCMotor motor1(1, MOTOR12_1KHZ); 
AF_DCMotor motor2(2, MOTOR12_1KHZ);
AF_DCMotor motor3(3, MOTOR34_1KHZ);
AF_DCMotor motor4(4, MOTOR34_1KHZ);
Servo myservo;   

boolean goesForward = false;
int distance = 100;
int speedSet = 0;

void setup() {

  myservo.attach(10);  
  myservo.write(115); 
  delay(2000);
  distance = readPing();
  delay(100);
  distance = readPing();
  delay(100);
  distance = readPing();
  delay(100);
  distance = readPing();
  delay(100);
}

void loop() {
 int distanceR = 0;
 int distanceL =  0;
 delay(40);
 
 if(distance <= 15)
 {
  moveStop();
  delay(100);
  moveBackward();
  delay(300);
  moveStop();
  delay(200);
  distanceR = lookRight();
  delay(200);
  distanceL = lookLeft();
  delay(200);

  if(distanceR >= distanceL)
  {
    turnRight();
    moveStop();
  }else
  {
    turnLeft();
    moveStop();
  }
 }else
 {
  moveForward();
 }
 distance = readPing();
}

int lookRight()
{
    myservo.write(50); 
    delay(500);
    int distance = readPing();
    delay(100);
    myservo.write(115); 
    return distance;
}

int lookLeft()
{
    myservo.write(170); 
    delay(500);
    int distance = readPing();
    delay(100);
    myservo.write(115); 
    return distance;
    delay(100);
}

int readPing() { 
  delay(70);
  int cm = sonar.ping_cm();
  if(cm == 0)
  {
    cm = 250;
  }
  return cm;
}

void moveStop() {
  motor1.run(RELEASE); 
  motor2.run(RELEASE);
  motor3.run(RELEASE);
  motor4.run(RELEASE);
  } 
  
void moveForward() {

 if(!goesForward)
  {
    goesForward = true;
    motor1.run(FORWARD);      
    motor2.run(FORWARD);
    motor3.run(FORWARD); 
    motor4.run(FORWARD);     
   for (speedSet = 0; speedSet < MAX_SPEED; speedSet +=2) 
   {
    motor1.setSpeed(speedSet);
    motor2.setSpeed(speedSet);
    motor3.setSpeed(speedSet);
    motor4.setSpeed(speedSet);
    delay(5);
   }
  }
}

void moveBackward() {
    goesForward = false;
    motor1.run(BACKWARD);      
    motor2.run(BACKWARD);
    motor3.run(BACKWARD);
    motor4.run(BACKWARD);  
  for (speedSet = 0; speedSet < MAX_SPEED; speedSet +=2) 
  {
    motor1.setSpeed(speedSet);
    motor2.setSpeed(speedSet);
    motor3.setSpeed(speedSet);
    motor4.setSpeed(speedSet);
    delay(5);
  }
}  

void turnRight() {
  motor1.run(FORWARD);
  motor2.run(FORWARD);
  motor3.run(BACKWARD);
  motor4.run(BACKWARD);     
  delay(500);
  motor1.run(FORWARD);      
  motor2.run(FORWARD);
  motor3.run(FORWARD);
  motor4.run(FORWARD);      
} 
 
void turnLeft() {
  motor1.run(BACKWARD);     
  motor2.run(BACKWARD);  
  motor3.run(FORWARD);
  motor4.run(FORWARD);   
  delay(500);
  motor1.run(FORWARD);     
  motor2.run(FORWARD);
  motor3.run(FORWARD);
  motor4.run(FORWARD);
}
